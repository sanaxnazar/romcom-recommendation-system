## 💕 About the Project

This project is a **movie recommendation system** built using Python and collaborative filtering.

It recommends romantic comedy movies based on **user rating patterns** from the MovieLens dataset.

Instead of recommending random movies, it identifies films that were enjoyed by users with similar tastes using **Cosine Similarity**.

✅ Data Cleaning

✅ Exploratory Data Analysis
